## Installation
Add this line to your application's Gemfile:

```ruby
gem 'ez-permissions'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install ez-permissions
```

## Generators

Generate ActiveRecord migrations:
```bash
rails generate ez:permissions:migrations
```
